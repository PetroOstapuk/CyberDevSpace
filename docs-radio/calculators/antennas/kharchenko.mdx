# Калькулятор антени Харченко (BiQuad)

:::info[Інформація]
Цей калькулятор використовується для розрахунку розмірів антени Харченко (зигзагоподібної BiQuad антени).
Антена Харченко є направленою антеною з високим підсиленням, що складається з двох квадратних рамок 
та металевого рефлектора. Вона широко використовується для Wi-Fi, GSM, LTE та інших застосувань.
:::

<Calculator />

:::warning[Примітки:]
- **Стандартні частоти:** Виберіть зі списку популярних частот або введіть власну частоту вручну.
- **Вхідний імпеданс:** Підтримуються стандартні значення 50 Ω та 75 Ω.
- **Одиниці вимірювання:** Результати можна відображати в міліметрах, сантиметрах або метрах.
- **Діапазон частот:** Калькулятор оптимізований для роботи в діапазоні 400-3000 МГц.
- **Рефлектор:** Виготовляється з металевого листа або дрібної металевої сітки.
- **Матеріал рамок:** Використовуйте мідний дріт відповідного діаметра (див. AWG розмір).
- **Точність виготовлення:** Для досягнення оптимальних характеристик дотримуйтесь розрахункових розмірів з точністю ±1мм.
- **Живлення:** Коаксіальний кабель підключається в центрі нижньої рамки з дотриманням розрахункового проміжку.
- **Поляризація:** Антена має лінійну поляризацію в площині рамок.
- **Підсилення:** Типове підсилення складає 10-12 dBi залежно від частоти та якості виготовлення.
- **Діаграма спрямованості:** Кут розкриття діаграми спрямованості становить приблизно 60-70 градусів.
- **Монтаж:** Рефлектор повинен бути розташований точно на розрахунковій відстані від площини рамок.
:::
