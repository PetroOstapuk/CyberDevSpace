# Калькулятор J-pole антени

:::info[Інформація]
Цей калькулятор використовується для розрахунку розмірів J-pole антени (J-антени).
J-pole антена є популярним типом вертикальної антени для VHF/UHF діапазонів.
:::

<Calculator />

:::warning[Примітки:]
- **Стандартні частоти:** Виберіть зі списку популярних частот або введіть власну частоту вручну.
- **Одиниці вимірювання:** Результати можна відображати в міліметрах, сантиметрах або метрах.
- **Розмір A** - довжина верхнього випромінюючого елемента (3/4 λ).
- **Розмір B** - довжина нижнього елемента (1/4 λ).
- **Відстань C** - відстань від точки живлення до нижнього кінця.
- **Розмір D** - додатковий розмір для налаштування.
- Коефіцієнт укорочення автоматично розраховується залежно від частоти та діаметра дроту.
- Діаметр дроту впливає на коефіцієнт укорочення та загальні розміри антени.
- Для кращих результатів використовуйте дріт діаметром 2-6 мм.
- Антена може бути виготовлена з дроту, трубки або смужки металу.
:::
