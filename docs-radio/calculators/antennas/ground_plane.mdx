# Калькулятор антени Ground Plane

:::info[Інформація]
Цей калькулятор використовується для визначення довжини кожного елемента антени Ground Plane.
Сюди входить вертикальний випромінювач та радіальні елементи.
:::

<Calculator />

:::warning[Примітки:]
- Довжина хвилі розраховується за стандартною формулою.
- Для розрахунку використовується чвертьхвильова конфігурація.
:::
