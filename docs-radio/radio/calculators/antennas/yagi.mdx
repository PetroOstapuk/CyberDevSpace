# Калькулятор Yagi антени (DL6WU)

:::info[Інформація]
Цей калькулятор використовується для розрахунку розмірів Yagi антени за методикою DL6WU.
Yagi-Uda антена є найпопулярнішим типом направленої антени для VHF/UHF діапазонів.
Калькулятор враховує корекцію на вплив буму (бумкорекція) та різні типи монтажу елементів.
:::

<Calculator />

:::warning[Примітки:]
- **Стандартні частоти:** Виберіть зі списку популярних частот або введіть власну частоту вручну.
- **Форма активного вібратора:** Петльовий диполь має кращу узгодженість (75 Ом), розрізний диполь має імпеданс ~50 Ом.
- **Вид елементу:** Круглі елементи простіші у виготовленні, плоскі мають дещо кращі характеристики.
- **Форма буму:** Круглий або квадратний переріз буму впливає на розрахунок корекції.
- **Монтаж елементів:**
  - **Варіант 1:** Елементи з'єднані з металічним бумом (найпростіший монтаж)
  - **Варіант 2:** Елементи ізольовані від буму (кращі характеристики)
  - **Варіант 3:** Діелектричний бум або віддалені елементи (найкращі характеристики)
- **Одиниці вимірювання:** Результати можна відображати в міліметрах, сантиметрах або метрах.
- Мінімальна кількість елементів - 3 (рефлектор + диполь + 1 директор).
- Максимальна кількість елементів обмежена 20 для практичних міркувань.
- Діаметр буму не повинен перевищувати 9% від довжини хвилі.
- Розрахунок ведеться з урахуванням бумкорекції за алгоритмом DL6WU.
- Підсилення розраховується за емпіричними формулами та є наближеним.
- Для точного налаштування рекомендується використовувати КСВ-метр та антенний аналізатор.
:::
