#  Калькулятор антени Flower Pot

:::info[Інформація]
Цей калькулятор використовується для розрахунку довжини кожного елемента коаксіальної антени типу **Flower Pot**.
У розрахунок входять довжина верхнього та нижнього плеча диполя, а також довжина кабелю,
необхідного для намотки запірного дроселя.
:::

<Calculator />

:::warning[Примітки:]
- Резонансний коефіцієнт - це значення між 5-10%, значення за замовчуванням - 7,5% або середнє значення.
- Коефіцієнт укорочення кабелю є постійною величиною, що залежить від типу його діелектрика. Для кабелів із поліетиленовим діелектриком, зокрема RG-58, він становить 0,66, а для кабелів із вспіненим діелектриком — 0,8.
- Кількість витків запірного дроселя не рахується, оскільки вона залежить від діаметра труби. Натомість розраховується довжина кабелю, необхідна для намотки.
- Намотка виконується за або проти годинникової стрілки, витки мають щільно прилягати один до одного. Початок і кінець намотки повинні точно відповідати розрахунковій довжині кабелю. Намотування здійснюється на ПВХ трубу діаметром 20–40 мм.
- Важливо! Труба не повинна мати металевого екрана всередині, оскільки це може порушити роботу дроселя.
:::
